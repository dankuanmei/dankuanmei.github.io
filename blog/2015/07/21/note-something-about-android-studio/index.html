Note Something About Android Studio

Jul 21st, 2015 9:22 pm

Create project with Google Plus login

Cần phải sửa lại phiên bản của play-services trong file build.gradle

compile 'com.google.android.gms:play-services:6.1.71'

Thao tác để enable Google Plus API

  1. Go to the Google Developers Console

  2. Click Create Project:

  3. In the Project name field, type in a name for your project, such as Quickstart App.

  4. In the Project ID field, optionally type in a project ID for your project or use the one that the console has created for you. This ID must be unique world-wide.

  5. Click the Create button and wait for the project to be created. Note: There may be short delay of up to 30 seconds before the project is created. Once the project is created, the name you gave it appears at the top of the left sidebar.

  6. In the left sidebar, select APIs & auth (the APIs sub-item is automatically selected).

  7. Find the Google+ API and set its status to ON—notice that this action moves Google+ API to the top of the list; you can scroll up to see it.
  8. In the sidebar under “APIs & auth”, select Consent screen.
  9. Choose an Email Address and specify a Product Name.
  10. In the left sidebar under “APIs & auth”, select Credentials.
  11. Click Create a new Client ID—the Create Client ID dialog box appears, as shown further below.
  12. Select Installed application for the application type.
  13. Select Android as the installed application type.

  14. Copy and paste the following package name for this quick-start app into the Package name field:

    com.google.android.gms.plus.sample.quickstart

  15. In a terminal, run the the Keytool utility to get the SHA-1 fingerprint of the certificate. For the debug.keystore, the password is android.

    keytool -exportcert -alias androiddebugkey -keystore -list -v
